GitHub est un service web d'hébergement : il permet de stocker des fichiers sur le Web.

Deplus GitHub propose également un service de gestion de développement de logiciels :

  • Il permet le suivi des bugs
  • La création d'une documentation
  • Le suivi des demandes de fonctionnalités

Allez sur le site https://github.com/signup et suivez les instructions pour créer un compte.

Sur GitHub un projet se nomme un repository (en français un dépôt).

Un repository peut contenir des dossiers et fichiers, des images, des vidéos etc...

Allez sur https://github.com/

En haut à droite d'une page, utilisez le menu déroulant + et sélectionnez Nouveau dépôt.

Tapez un nom pour votre dépôt et une description facultative.

Laisser les autres paramètres par défaut puis en bas à droite cliquer sur "Créer le dépôt"

HTTP est un protocole de communication entre un client et un serveur web.

Pour rappel voici son principe de fonctionnment :

Nous allons créer une requête HTTP puis l'envoyer au serveur HTTP et analyser sa réponse.

Création de la requête

Rendez-vous sur le site httpie et cliquez sur Try online

Vous aboutissez devant un prompt. Appuyez sur enter.

Vous pouvez maintenant saisir une requête et examiner son contenu ainsi que sa réponse.

Une requête (http -v PUT pie.dev/put API-Key:foo hello=world) est préremplie. Supprimer la en appuyant sur la touche effacer.

Taper la commande : 

http -v GET <URL de votre site Web>

Vous remplacez par l'URL de votre site que vous avez oublié dans les étapes précédentes.

Analyse de la requête

On exécute la commande "http" afin d'indiquer qu'on souhaite créer une requête HTTP.

L'option -v pour "verbose", verbeux affiche un contenu détaillé

La méthode HTTP GET permet d'accéder à un contenu, sans le modifier.
